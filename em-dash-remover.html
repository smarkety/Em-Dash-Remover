Em Dash Remover

One click removes em dashes from any webpage, intelligently replacing them with the right punctuation based on context. Works on any page.

Copy the Bookmarklet Code

Copy the code below, then follow the install steps to add it as a bookmark.

// Em Dash Remover – smarkety javascript:(function(){ function r(t) { // Before punctuation: drop the dash, keep the punctuation t = t.replace(/\s*[\u2014\u2013]\s*([,\.;:])/g, '$1'); // After sentence end (.!?): space only t = t.replace(/([\.!?])\s*[\u2014\u2013]\s*/g, '$1 '); // Between two words: comma + space t = t.replace(/(\w)\s*[\u2014\u2013]\s*(\w)/g, '$1, $2'); // Fallback: spaced hyphen t = t.replace(/[\u2014\u2013]/g, ' - '); return t; } function walk(n) { if (n.nodeType === 3) { var o = n.textContent, c = r(o); if (o !== c) n.textContent = c; } else { n.childNodes.forEach(walk); } } var b = document.body.innerHTML; walk(document.body); var n = (b.match(/[\u2014\u2013]/g) || []).length; var t = document.createElement('div'); t.style.cssText = 'position:fixed;bottom:1.5rem;right:1.5rem;background:#0d0d0d;color:#ade6b9;border:1px solid #467061;padding:.75rem 1.25rem;border-radius:.5rem;font:500 .85rem/1 monospace;z-index:999999;box-shadow:0 4px 20px rgba(0,0,0,.7)'; t.textContent = n ? '\u2713 '+n+' em dash'+(n>1?'es':'')+' removed' : 'No em dashes found'; document.body.appendChild(t); setTimeout(function(){ t.remove(); }, 3000); })();
Install the Bookmarklet
  • 1Create a new bookmark in your browser.
  • 2Set the name to Em Dash Remover.
  • 3Paste the copied code from Step 1 into the URL field.
  • 4Save. The bookmarklet is now ready to use on any page.
How to Use
  • 1 Open any webpage: your own site, a client's CMS, WordPress, Shopify, whatever.
  • 2 Click Em Dash Remover in your bookmarks bar. All em dashes are replaced instantly.
  • 3 A small popup confirms how many were replaced. Changes are visual only and the actual page source is not modified. Copy cleaned text manually or use it for review.
Smart Replacement Logic
Context Replaced with Example
Before punctuation → removed great—, → great,
After sentence end → space done.—Then → done. Then
Between two words → comma good—great → good, great
En dash (–) → same rules U+2013 treated identically
Fallback → - anything else → spaced hyphen
